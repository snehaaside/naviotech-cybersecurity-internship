## Projects

### 1. Minor Project 1 — Cybersecurity Risk Assessment Framework for Small Businesses
A NIST CSF-aligned, 7-stage risk assessment model for SMEs, using a 5×5 Likelihood × Impact scoring matrix. Validated across three sector case studies (retail, fintech, healthcare) with an average risk score reduction of 62%.
→ [`/01-risk-assessment-framework`](./01-risk-assessment-framework)

### 2. Minor Project 2 — Zero Trust Architecture for Enterprise Security
A simulated Zero Trust implementation covering Adaptive MFA, RBAC, and network micro-segmentation, tested in a VMware/Docker lab environment against insider-threat and external-breach scenarios.
→ [`/02-zero-trust-architecture`](./02-zero-trust-architecture)

### 3. Major Project — Phishing Awareness Simulation Using Social Engineering Techniques
A controlled, consent-based phishing simulation exercise ("NexoraCorp") measuring human-factor security risk through click-through and reporting rates.
→ [`/03-phishing-awareness-simulation`](./03-phishing-awareness-simulation)
