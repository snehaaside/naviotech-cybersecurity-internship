# Zero Trust Architecture for Enterprise Security

**Type:** Minor Project 2  
**Author:** Kumari Sneha

An enterprise-grade Zero Trust Architecture (ZTA) framework built around "Never Trust, Always Verify." Covers Adaptive MFA, cryptographic RBAC, and network micro-segmentation, simulated in a VMware/Docker lab environment against insider-threat and external-breach scenarios. Micro-segmentation contained simulated blast radius to 10% of assets, versus full compromise in a traditional flat-network model.

See `minor-2_merged.pdf` for the full report.
