# Phishing Awareness Simulation Using Social Engineering Techniques

**Type:** Major Project  
**Author:** Kumari Sneha  
**Organization:** Naviotech Solution Pvt Ltd

## Overview

A controlled, consent-based social engineering exercise conducted under the fictitious entity "NexoraCorp," designed to measure human-factor security risk within an organization. The project simulates a realistic phishing campaign — from email design to mock credential capture — while maintaining strict ethical boundaries: no real credentials were collected, and all participants provided informed consent prior to the exercise.

## Methodology

1. **Research** — Identified common phishing pretexts and psychological triggers (urgency, authority, trust)
2. **Email Design** — Crafted a realistic phishing email using the NexoraCorp fictitious brand
3. **Mock Login Page** — Built a simulated login page to safely record click-through and interaction behavior without capturing real credentials
4. **Informed Consent** — All participants were briefed and consented before the simulation ran
5. **Data Collection** — Response behavior was logged and analyzed over a 5-day window

## Key Results

- **49%** of participants clicked the link or entered data on the mock login page
- **22%** proactively reported the email as suspicious
- Click-through rates declined and reporting rates rose over the 5-day window as awareness spread organically
- Urgency and authority cues were the strongest psychological triggers observed
- Department-level differences in susceptibility were noted
