# Cybersecurity Risk Assessment Framework for Small Businesses

**Type:** Minor Project 1  
**Author:** Kumari Sneha

A practical, low-cost cybersecurity risk assessment framework for SMEs, grounded in the NIST Cybersecurity Framework. Includes a 7-stage risk assessment process, a 5×5 Likelihood × Impact risk-scoring matrix, mapped mitigation controls, and validation across three case studies (retail, fintech, healthcare) showing an average 62% risk reduction.

See `Cybersecurity_Risk_Assessment_Framework_Report.pdf` for the full report.
